# 🌸 PerfumeIQ AI

AI-Powered Personalized Fragrance Recommendation Agent built for the DSN X BCT LLM Agent Challenge.

## Features
- Personalized fragrance recommendations
- AI-powered perfume assistant
- Pricing intelligence dashboard
- Revenue & profit analytics
- Customer preference insights
- Inventory intelligence

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- OpenRouter API

## Project Structure
- `app.py` → Main Streamlit application
- `perfume_sales.csv` → Sales dataset
- `perfume_pricing_analysis_updated.xlsx` → Pricing intelligence data

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
